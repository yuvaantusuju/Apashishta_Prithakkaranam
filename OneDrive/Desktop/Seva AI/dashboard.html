Download Model Files

If you want to run the AI locally, download the trained model assets below and place them alongside:

  • model.json
  • metadata.json
  • weights.bin
How to Load Your Model

Once you have downloaded the three model files, follow these steps to load them into the app:

1
Click the Model Status Badge

Look at the top-right corner of the app. Click on "Load Model Files" in the status badge.

2
Select model.json

A file picker will open. Choose the downloaded model.json file from your device.

3
Select metadata.json

After selecting model.json, the app will ask for metadata.json. Choose it from your downloads.

4
Select weights.bin

Finally, select the weights.bin file. The model will load and you'll see "Model ready ✓" status.

💡 Tip: The model files must be selected in the correct order: model.json → metadata.json → weights.bin. The app will guide you through each step. Make sure all three files are from the same model download to avoid compatibility issues.

Project Description

Kachara-vibhajanam (కచరా విభజనం) helps individuals and communities correctly identify and sort waste using machine learning. Point your camera at any waste item — the app classifies it into one of six categories and provides clear disposal guidance.
